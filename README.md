# Glossy Belle - UGC Creator Portfolio

A premium, high-converting portfolio website designed for UGC (User Generated Content) creators to showcase their creative work, brand collaborations, and professional services. Built with a focus on aesthetics, speed, and conversion.

## ✨ Features

- **Dynamic Video Showcase**: An interactive Instagram Reels gallery that fetches thumbnails dynamically to display real-time content.
- **Brand Showcase**: A dedicated section to highlight previous collaborations and trusted brands.
- **Service breakdown**: Detailed overview of content creation services, from aesthetic product shoots to authentic reviews.
- **Modern UI/UX**: A "Glossy" and vibrant aesthetic using a sophisticated pink and red color palette.
- **Fully Responsive**: Optimized for a seamless experience across mobile, tablet, and desktop devices.
- **Performance Focused**: Built with Next.js 15 for lightning-fast load times and SEO optimization.

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## 🛠️ Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **View the project**: Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `src/components/`: Modular UI components like `Hero`, `InstaVideos`, and `BrandShowcase`.
- `src/app/`: Next.js App Router configuration and global styles.
- `public/`: Static assets, brand logos, and fallback images for the video gallery.
